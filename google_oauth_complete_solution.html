🚨 Problem Diagnosis

Error: "SyntaxError: Unexpected token '<', "<!DOCTYPE "... is not valid JSON"

Root Cause: Your frontend code is trying to parse HTML as JSON. This happens when:
  • You're calling a web OAuth endpoint expecting JSON
  • The OAuth callback returns HTML instead of JSON
  • CORS issues causing error pages to be returned

✅ Solution 1: Use Web Flow (Recommended)

This is the standard OAuth flow that works with most frontend frameworks:

JavaScript Implementation:

function loginWithGoogle() { // Option 1: Direct redirect (for regular web apps) window.location.href = 'http://localhost:8000/accounts/google/login/'; // Option 2: Open in popup (for SPAs) const popup = window.open( 'http://localhost:8000/accounts/google/login/', 'google_login', 'width=500,height=600,scrollbars=yes,resizable=yes' ); // Listen for popup to close or redirect const checkClosed = setInterval(() => { if (popup.closed) { clearInterval(checkClosed); // Check if user is now logged in checkLoginStatus(); } }, 1000); } async function checkLoginStatus() { try { const response = await fetch('http://localhost:8000/api/google/success/', { credentials: 'include' }); if (response.ok) { const data = await response.json(); console.log('Login successful:', data); // Handle successful login handleLoginSuccess(data); } } catch (error) { console.log('Not logged in yet'); } } function handleLoginSuccess(data) { // Store the token localStorage.setItem('auth_token', data.token); // Update UI document.getElementById('user-info').innerHTML = ` Welcome, ${data.user.first_name} ${data.user.last_name}!
Email: ${data.user.email} `; // Hide login button, show logout button document.getElementById('login-btn').style.display = 'none'; document.getElementById('logout-btn').style.display = 'block'; }

✅ Solution 2: API Endpoint (For Mobile/SPA with Token)

Use this only if you have a Google access token from Google OAuth SDK:

// Only use this if you have a Google access token async function loginWithGoogleToken(googleAccessToken) { try { const response = await fetch('http://localhost:8000/api/google/login/', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ access_token: googleAccessToken }) }); if (response.ok) { const data = await response.json(); console.log('Login successful:', data); return data; } else { const error = await response.json(); console.error('Login failed:', error); throw new Error(error.detail || 'Login failed'); } } catch (error) { console.error('Network error:', error); throw error; } }
