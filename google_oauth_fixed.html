üìù Frontend Integration Examples

For Regular Web App (HTML/JavaScript):

// Simple redirect to Google OAuth window.location.href = 'http://localhost:8000/accounts/google/login/'; // Or open in popup window.open('http://localhost:8000/accounts/google/login/', 'google_login', 'width=500,height=600');

For SPA (React/Vue/Angular):

// Method 1: Get OAuth URL first const response = await fetch('http://localhost:8000/api/google/url/'); const data = await response.json(); window.location.href = `http://localhost:8000${data.login_url}`; // Method 2: Direct redirect window.location.href = 'http://localhost:8000/accounts/google/login/';

For Mobile App (with access token):

// After getting access token from Google OAuth SDK const response = await fetch('http://localhost:8000/api/google/login/', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ access_token: 'your_google_access_token_here' }) }); const result = await response.json();

üêõ Common Issues & Solutions

Issue: "Unexpected token '<', "<!DOCTYPE "... is not valid JSON"

Cause: Your frontend is trying to parse HTML as JSON

Solutions:

  • Use Method 1 (Web Flow) for regular web apps
  • Use Method 2 (API Endpoint) only if you have a Google access token
  • Check that you're using the correct endpoint URL
  • Ensure proper CORS settings if calling from different domain

Issue: OAuth redirects to wrong URL

Solution: Update your Google OAuth app settings to include:

  • http://localhost:8000/accounts/google/login/callback/
  • http://127.0.0.1:8000/accounts/google/login/callback/
